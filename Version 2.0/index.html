How Strong Is Your Password?

This project is solely only for educational purposes, We do not keep nor store your passwords.

This project was created to allow people test the strength of there passwords, we use a list of "ban passwords" that will auto-bring your password to 0% this is a indicator, that your password is either leaked, or commonly used word. The auto 0% also means you have repeted character which we do not recommend.
